    - Nome

    - Descrição
    - Estoque
    - Status
 - [ ] CRUD of Pedido
    - Codigo do Pedido
    - Cliente

    - Valor do Frete
    - Total
    - Data de Criação
    - Forma de Pagamento
    - Forma de Entrega

    - Codigo do Pedido
    - COdigo do Produto
    - Nome do Produto
    - Quantidade
    - Oreço Unitário
    - Total
 - [ ] Autenticação JWT

 - [ ] Pagamento a vista tem 5% de desconto
 - [ ] Pagamento a prazo deve ser informado o numero de parcelas
 - [ ] Rota para acessar todos os pedidos realizados pelo cliente.
